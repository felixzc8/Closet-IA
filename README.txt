Author: Felix Chen
Title: Closet App

Dependencies:
This app uses AmbilWarnaDialog by yukuku
Instructions to integrate the AmbilWarnaDialog dependency on the Github page
https://github.com/yukuku/ambilwarna/blob/master/library/src/main/java/yuku/ambilwarna/AmbilWarnaDialog.java

How to run:
To run the app, expand the zip file found in the Product folder. This will produce a folder called ClosetIA

Download and install Android Studio from: https://developer.android.com/studio

Once Android Studio has been installed, open the ClosetIA folder with Android Studio.

The app should be able to run on any emulated Android Device using Android 7.2 or higher. It is recommended to use a Pixel 5 API 32.

Once a device has been selected, the app can be run.
